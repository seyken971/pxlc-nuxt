# Corpus de référence — enfants et écrans

Conversions markdown de documents publics officiels, utilisées pour sourcer
toute affirmation santé ou usage des écrans publiée sur le site (règle
CLAUDE.md — ne jamais inventer un chiffre ou une recommandation, citer le
document et la section).

Règle de citation — les recommandations se citent depuis les **avis**
(ex. avis HCSP 12/12/2019, section VII pour les recommandations
parents/encadrants) ; les données chiffrées et l'analyse de la littérature
se vérifient dans les **rapports**.

Les noms `hcspa*`/`hcspr*` reprennent la nomenclature officielle des PDF du
HCSP (a = avis, r = rapport). Le communiqué de presse HCSP (`hcspc*`) a été
volontairement écarté du corpus — on cite l'avis, pas sa vulgarisation.

Documents administratifs français librement rediffusables avec attribution ;
ce dossier n'est pas publié sur le site (seul `.output/public` est déployé).
