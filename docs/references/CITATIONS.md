# Carte de citation — docs/references/

Quel document ouvrir pour sourcer quelle affirmation.
Pour chaque claim : .md (section) et PDF (page) — les numéros de page sont
ceux de la table des matières des PDFs originaux.

## Règles d'usage

- Toute affirmation santé/écrans → citer document + section (et page PDF si disponible)
- Toujours citer « HCSP 2019-2020 · HAS 2020 » ensemble pour le cadre réglementaire
- Jamais « addiction » — dire « usage problématique des écrans »
  (fondement : hcspa20210308 § 4.2, p. 13)
- Recommandations parentales → citer l'**avis** (hcspa*) ; données chiffrées
  et revue de littérature → citer le **rapport** (hcspr*)
